# GAMES202 homework1

## Usage

### For Visual Studio Code
Install plugin `Live Sever` and run with `index.html` directly

### For Node.js users
To install:
```
npm install http-server -g
```
To run(from `index.html` directory):
```
http-server . -p 8000
```
